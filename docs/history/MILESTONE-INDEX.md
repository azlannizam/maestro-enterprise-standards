# Maestro Milestone Index

This index is the authoritative register of historical Maestro milestones.
Milestones record completed events that prove or change how the Maestro
Enterprise Operating System operates.

Milestones complement MES standards and MADs:

- MES documents define binding standards and requirements.
- MADs record permanent architecture and governance decisions.
- Milestones record completed historical events and their evidence chain.
